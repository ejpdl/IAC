About IAC Monitoring System

The IAC Monitoring System is designed to efficiently monitor and manage student usage of computers in the Internet Access Control (IAC) Laboratory. It ensures that computers are used primarily for academic and research purposes while providing real-time data and insights into student activity.

The system includes a mobile application for students, allowing them to easily check the availability of computers. Students can view whether a computer is available, occupied, or pending request. This makes it convenient for users to plan their time in the lab efficiently.

On the administrative side, the system features a web-based platform that helps administrators manage student requests, track session histories, and monitor overall lab usage. The admin interface also includes a dashboard with graphs, diagrams, and charts to provide detailed analytics on computer usage, helping to improve lab management and ensure fair access for all students.

Overall, the IAC Monitoring System streamlines the process of managing computer resources, improves access control, and offers valuable data for analysis and system optimization.
